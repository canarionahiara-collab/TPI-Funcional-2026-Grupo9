## 🎯 Objetivo del Proyecto

Desarrollar un **Sistema de Semáforos Inteligentes** aplicando los principios del paradigma funcional mediante Common Lisp.

El sistema implementa:

- Estados y transiciones semafóricas.
- Temporización automática basada en tiempo Unix.
- Sistema de auditoría y registro de eventos.
- Análisis de ciclos semafóricos.
- Planificación temporal.
- Integración de bibliotecas externas mediante Quicklisp.
- Estudio comparativo con otro lenguaje funcional.
